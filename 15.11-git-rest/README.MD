# REST
## Rest is Representational State Transfer


# Requests type:
# GET, POST, PUT, PATCH, DELETE.

- GET - GET data
- POST - Send data
- PUT - Send all object
- PATCH - Sends one property of an object
- DELETE - Delete an object.
